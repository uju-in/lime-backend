**프로젝트를 새로 만드는 이유**
1. 기획 아이디어가 좋아서 실제 서비스 운영을 해보고 싶음
2. 기존 경험을 살려 새로운 팀원들과 협업 경험을 쌓아 나가고 싶음
3. 리팩토링 및 리엔지니어링으로 프로젝트 보완(기능이 부족해서 기능을 보완하고자 함) 및 최적화
